BMad Method (**B**reakthrough **M**ethod of **A**gile AI **D**riven Development) 是一个 AI 驱动的开发框架，帮助你更快、更智能地构建软件。它提供专门的 AI Agents、引导式 Workflows 以及能够适应项目复杂度的智能规划——无论你是修复 Bug 还是构建企业级平台。

如果你习惯使用 Claude、Cursor 或 GitHub Copilot 等 AI 编程助手，那么你已经准备好开始了。

---

## 你需要什么

BMad 适用于任何支持自定义 System Prompts 或项目上下文 (Context) 的 AI 编程助手。流行的选择包括：

- **[Claude Code](https://code.claude.com)** — Anthropic 的 CLI 工具（推荐）
- **[Cursor](https://cursor.sh)** — AI 优先的代码编辑器
- **[Windsurf](https://codeium.com/windsurf)** — Codeium 的 AI IDE
- **[Roo Code](https://roocode.com)** — VS Code 扩展

你应该熟悉基本的软件开发概念，如版本控制 (Version Control)、项目结构和敏捷 (Agile) 工作流。不需要有 BMad 风格 Agent 系统的先前经验——这正是本文档的用途所在。
